CADOP Service

Welcome to the CADOP (Custodian-Assisted DID Onboarding Protocol Service) Service. This service provides OIDC-based authentication and DID operations.

Available Endpoints:

OIDC Discovery
/.well-known/openid-configuration
JWKS
/.well-known/jwks.json
DID Document
/.well-known/did.json
Authorization Endpoint
/auth/authorize
Token Endpoint
/auth/token
UserInfo Endpoint
/auth/userinfo
WebAuthn API
/api/webauthn/*
Custodian API
/api/custodian/*
Proof API
/api/proof/*

For more information, please refer to the documentation.
