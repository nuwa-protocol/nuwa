MCP specifications supports [returning resources](https://modelcontextprotocol.io/specification/2025-06-18/server/tools#embedded-resources) along with a tool result. We leveraged this mechanism and defined a resource schema for UI resources. When a tool returns a UI resource, the client will render an iframe that embeds the UI described by this UI resource.

## UI Resource Schema

Here's the UI resource schema:

```ts
interface UIResource {
  uri: string;                  // capui:/<path>
  name: string;                 // display name
  description?: string;
  mimeType: 'text/x-nuwa-capui-path';
  text: string;                 // path
}
```
