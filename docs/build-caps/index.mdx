## What You'll Build

Caps let you package an AI capability as a single shareable unit: Prompt + Model + optional MCP tools + optional UI artifact. This section walks you through composing, testing, and publishing a Cap.

## Prerequisites

- Node.js 18+ and pnpm
- A DID (created via Identity Kit or CADOP)
- Optional: access to the Nuwa LLM Gateway or your own compatible gateway

## Quick Start

Install the TypeScript SDK packages you need:

```bash
pnpm add @nuwa-ai/cap-kit @nuwa-ai/identity-kit @nuwa-ai/payment-kit
```

Register a Cap programmatically:

```ts
import { CapKit } from '@nuwa-ai/cap-kit';
import { TestEnv, createSelfDid } from '@nuwa-ai/identity-kit/testHelpers';

// 1) Identity & signer
const env = await TestEnv.bootstrap({ rpcUrl: 'https://test-seed.rooch.network', network: 'test' });
const { signer, did } = await createSelfDid(env);

// 2) CapKit client
const capKit = new CapKit({
  roochUrl: 'https://test-seed.rooch.network',
  mcpUrl:   'https://<your-mcp-server>',
  contractAddress: '<cap-registry-contract>',
  signer,
});

// 3) Compose & register
await capKit.registerCap({
  authorDID: did,
  idName: 'my_first_cap',
  core: {
    prompt: { value: 'You are a helpful assistant. {{user_geo}}', suggestions: ['Summarize this...'] },
    model: {
      providerId: 'openrouter',
      modelId: 'openai/gpt-4o-mini',
      supportedInputs: ['text'],
      contextLength: 128000,
    },
    mcpServers: { tools: 'https://<your-mcp-server>' },
    // artifact: { srcUrl: 'https://your-cap-ui.example.com' },
  },
  metadata: {
    displayName: 'My First Cap',
    description: 'Example Cap built with Nuwa',
    introduction: 'What this Cap does and how to use it...',
    tags: ['example', 'starter'],
  },
});
```
